Archived / Validated Idea 05

Design Knowledge Agent

An enterprise LLM agent that integrates Figma files, code repository tokens, and research documents to serve as the unified designer resource hub.

Observation & Context

Designers and developers spend dozens of hours searching for style patterns, figma components, tokens, guidelines, and previous research findings. This siloing leads to fragmented products and duplication of design work.

Research

Internal audits show up to 30% project delays and code rewrites because developers construct custom styling utilities instead of identifying the appropriate existing components.

Archival Retrospective

While the core problem of "Organizational Amnesia" remains a fascination, we archived this project because the space has matured rapidly, with multiple third-party SaaS products offering direct Figma-to-code retrieval. We shifted focus to more blue-ocean hypotheses.
